CS 239 — Unix Systems Programming

Homework 3

Focus:

  • Process Control

Task:

Write a simple shell. It should accept commands, including arguments and execute them.

Environment

  • Use whatever prompt string you like by default, however note that it should end in a space otherwise it won't look very good on the screen.
  • If the PS1 environment variable is set, then use that instead. Don't worry about "special" characters in the prompt string, such as \d or \h.

Built-in Commands

  • exit: Your shell should exit when the user types exit.

Suggestions

  • Use strtok to tokenize the command line.

Not included

Do not attempt at this point:

  • Piping
  • I/O redirection
  • Background processing
  • Globbing
  • Quotes

Turn in:

  • For this and all future assignments:
    • provide a Makefile.
    • provide a README if there is anything I need to know about your program, e.g. it doesn't work.
    • Submit your solution on Blackboard as a compressed tar file.
