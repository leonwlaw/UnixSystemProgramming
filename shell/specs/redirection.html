CS 239 — Unix Systems Programming

Homework 4

Focus:

  • I/O redirection
  • Not Piping

Task:

Add to your shell:

  • I/O redirection.
    • > and <.
    • You may assume that they are surrounded by whitespace.
    • We are only redirecting from standard in and to standard out. No other file descriptors need be supported. Nor is appeding supported.
    • I/O redirection may occur anywhere in the command, not just at the end. Experiment with the bash shell if you are not sure what can and cannot be done.
  • Do not use variable-sized local arrays. If an array is defined locally, its size should be known at compile time. Otherwise, use the heap.

Not included

Do not attempt at this point:

  • Piping
  • Background processing
  • Globbing
  • Quotes

Turn in:

Submit your solution on Blackboard as hw04.zip. It should include

  • A makefile
  • The source hw04.c
  • Optional README. If there is anything I should know about, such as features that don't work.
